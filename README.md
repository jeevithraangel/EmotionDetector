# Final Project - Emotion Detector

## Project Description
This is a Final Project - Emotion Detector web application built using Python, Flask, and Watson NLP. It detects emotions like joy, sadness, anger, fear, and disgust from user input text.

## Features
- Emotion detection from text
- Flask web application
- Watson NLP integration
- Error handling for invalid input
- Static code analysis support
