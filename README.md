# Emotion Detector

## Project Description

Emotion Detector is a web application developed using Python and Flask. The application analyzes user text and identifies emotions such as Joy, Sadness, Anger, Fear, and Disgust.

## Features

* Detects Joy
* Detects Sadness
* Detects Anger
* Detects Fear
* Detects Disgust
* Handles multiple emotions
* Detects emotion ties
* Displays the dominant emotion
* User-friendly web interface

## Technologies Used

* Python
* Flask
* HTML
* JavaScript

## Installation

1. Install Flask:

pip install flask

2. Run the application:

python server.py

3. Open your browser and visit:

http://127.0.0.1:5000
